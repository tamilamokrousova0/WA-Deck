# WA Deck v2

Electron-приложение для macOS (Apple Silicon) с WhatsApp Web:
- 1 аккаунт на старте + добавление новых (`WP_x`, новый цвет иконки),
- перевод сообщений через DeepL API (авто + всплывающее окно с выбором языка),
- отложенная отправка при открытом приложении (текст + вложения).

## Быстрый старт

```bash
npm install
npm start
```

## Настройка DeepL Translate

1. Откройте панель справа.
2. Вставьте `DeepL API key` (для бесплатного тарифа обычно заканчивается на `:fx`).
3. Сохраните настройки.
4. Для автоперевода включите режим `Авто`.

## Сборка DMG (M-чипы)

```bash
npm run dist:mac
```

Результат появится в папке `dist/`.

## Важные замечания

- Отложенная отправка работает, пока приложение запущено.
- Вложения в WhatsApp Web отправляются best-effort через DOM, стабильность зависит от версии интерфейса WhatsApp.
- Для автоперевода переводятся только входящие видимые сообщения активной вкладки.
